What is Loose equality

It compares two values after converting their data types automatically

What is strict equality

It compares both value & datatype

WHat are the differences loose and strict quality.Which should be used and why.

    Loose Equality ==

  • Checks Only Value
  • Converts datatype automatically
  • Can give unexpected results
  • less Safe

    Strict Equality ===

  • Checks Value and datatype
  • No datatype Conversion
  • Can give accurate results
  • More Safe
